# Animations & UI Feedback

A simple website that shows different animation effects and interactive UI elements. It's a great way to learn and explore how animations can improve user experience.

## What's Inside?

### Built-in Animations
This project displays 4 common animations:
- **Spin** - A rotating loader (useful for loading indicators)
- **Ping** - A pulsing notification effect (great for alerts)
- **Pulse** - A fading in and out effect (perfect for skeleton loaders)
- **Bounce** - A bouncing motion (good for attention grabbers)

### Button Animations
Interactive buttons that respond when you hover over them:
- **Scale Effect** - Button grows when you hover
- **Shadow Effect** - Button glows with a shadow on hover

## How to run in VS Code

1. Open this folder in VS Code.
2. Install the Live Server extension if you do not already have it.
3. Open `index.html`.
4. Right-click inside the editor and select **Open with Live Server**.

The page will open in your default browser, and you can see the project live.

## Files Explained

- `index.html` - The website structure (what you see)
- `input.css` - The styling rules (colors, animations, layouts)
- `output.css` - The generated CSS file (created after building)
- `package.json` - Project settings and dependencies
